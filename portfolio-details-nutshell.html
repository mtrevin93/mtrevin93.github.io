Nutshell

Nutshell, an app for posting and sharing articles, events, tasks, and private/public messages with friends. Nutshell is a ReactJS CRUD app group project for NSS. I was personally responsible for creating the events page with affordance to view current weather and weather for events scheduled in a 5-day forecast window as well as the friends page.
